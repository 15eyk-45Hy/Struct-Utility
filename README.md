# Struct-Utility

## Structure and OOP simulation in Batch

This repository demonstrates how to simulate complex data structures and Object-Oriented Programming (OOP) concepts within standard Windows Batch files (`.bat` / `.cmd`). 

Since Batch lacks native support for classes and structures, this project utilizes advanced variable expansion (`enabledelayedexpansion`) and dynamic variable naming to implement pseudo-objects, constructors, and encapsulation.
